    <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-3">
      Quick Start
    </h2>
    <p className="text-base text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get up and running with GAOTEV in minutes
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Clone the GAOTEV repository to your local machine:
      
      ```bash
      git clone https://github.com/Gusivelin/GAOTEV.git
      cd GAOTEV
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install the required npm packages:
      
      <CodeGroup>
      ```bash npm
      npm install
      ```
      
      ```bash yarn
      yarn install
      ```
      
      ```bash pnpm
      pnpm install
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Configure environment variables">
      Create a `.env` file in the root directory with your database and authentication settings:
      
      ```bash .env
      DATABASE_URL="postgresql://user:password@localhost:5432/gaotev"
      DIRECT_URL="postgresql://user:password@localhost:5432/gaotev"
      NEXTAUTH_SECRET="your-secret-key-here"
      NEXTAUTH_URL="http://localhost:3000"
      ```
      
      <Note>Generate a secure `NEXTAUTH_SECRET` using `openssl rand -base64 32`</Note>
    </Step>
    
    <Step title="Run database migrations">
      Set up your database schema using Prisma:
      
      ```bash
      npx prisma migrate dev
      ```
      
      This will create the necessary database tables based on the schema defined in `prisma/schema.prisma`.
    </Step>
    
    <Step title="Start the development server">
      Launch the application in development mode:
      
      ```bash
      npm run dev
      ```
      
      Open [http://localhost:3000](http://localhost:3000) in your browser to see the login page.
    </Step>
  </Steps>
</div>
